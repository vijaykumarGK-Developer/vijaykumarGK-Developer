## 📂 Featured Projects

### 🩸 [Rakta-Vahini — Blood Donation Network](https://github.com/vijaykumarGK-Developer/raktavahini-web)

A cross-platform blood donation ecosystem connecting donors with emergency requests in real time.

| Platform | Stack | Features |
|---|---|---|
| **Web App** ([repo](https://github.com/vijaykumarGK-Developer/raktavahini-web)) | React 19, TypeScript, Vite, Tailwind CSS, shadcn/ui, Firebase, Leaflet | Interactive map, smart donor scoring, SOS broadcast, dark mode, CI/CD |
| **Android App** ([repo](https://github.com/vijaykumarGK-Developer/rakthavahini)) | Kotlin, Jetpack Compose, MVI, Firebase Auth, Firestore | Native mobile experience, real-time sync, offline support |
| **HTML/JS** ([repo](https://github.com/vijaykumarGK-Developer/rakthavahini-html)) | Vanilla JavaScript, Leaflet, Firebase | Lightweight browser version, Haversine distance calculation |

**Key algorithms:** Haversine distance formula, composite smart scoring (`distance + (responseSpeed × 0.1) + (activeDaysAgo × 0.5) − (freq × 0.2)`), 90-day eligibility check.

### 📈 [Delta Crypto Trading View — Backtest Strategy](https://github.com/vijaykumarGK-Developer/delta-crypto-trading-view-backtest-stratergy)

A Python-based algorithmic trading framework using backtrader. Implements 7-block notebook architecture with 3 trading strategies (Golden Cross, RSI Mean Reversion, MACD Divergence), performance analytics, scalping optimization, and automation templates.

### 📦 [OmniStock Inventory Management](https://github.com/vijaykumarGK-Developer/OmniStock-Inventory-Management)

MIT-licensed inventory management system built with HTML, CSS, and JavaScript.
